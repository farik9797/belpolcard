# БелПолКард — пластиковые карты

Pixel-perfect адаптивная вёрстка лендинга по макету Figma «All cards/1440».
Чистый HTML + CSS, без сборщиков и зависимостей.

## Запуск

```bash
python3 -m http.server 4321
```

Затем откройте <http://localhost:4321>.

## Структура

```
index.html          главная — «Пластиковые карты», 11 секций
rfid.html           «RFID-продукция» — карточки, браслеты, брелоки
card-discount.html  карточка товара «Скидочные карты»
pos.html            «POS-материалы» — собрана из дизайн-системы (в Figma
                    этой страницы нет); медиа-блоки карточек — плейсхолдеры
styles.css          стили, токены из макета
app.js              бургер-меню, галерея товара, формы
assets/             изображения из Figma (webp)
fonts/              Lato и Jost (woff2, self-hosted)
```

Шапка и футер продублированы в каждой странице (статический сайт без
шаблонизатора); общий JS вынесен в `app.js`. Внутренние страницы
использовали в макете шрифты UNCAGE и Gilroy — заменены на Lato/Jost по
ролям (UNCAGE — капсовый display-шрифт → Lato + uppercase).

## Как это устроено

**Сетка.** Базовая ширина макета — 1440px: контейнер 1200px, поля по 120px.
Реализовано как `width: min(1200px, 100% - 2 * gutter)` — на 1440px даёт ровно
макетные значения, ниже начинает сжиматься.

**Типографика резиновая.** Все размеры на `clamp()` с линейным ростом между
768px и 1440px. Формула подобрана так, что на 1440px значения в точности равны
макетным (56 / 48 / 20 / 16px), а на 768px садятся на минимум (32 / 28 / 16 / 14px).
Отдельных `font-size` в медиазапросах нет.

**Шрифты локальные**, внешних запросов нет. Lato 2.015 и Jost 3.710, оба под
[SIL OFL 1.1](https://scripts.sil.org/OFL) — тексты лицензий лежат в `fonts/`.
Взят именно официальный Lato 2.0, а не версия из Google Fonts: в последней
отсутствует начертание Medium (500), которое используется в макете.
Сабсет latin + cyrillic, суммарно ~134 KB на шесть начертаний.

Пересобрать шрифты (скачает исходники сам):

```bash
pip install fonttools brotli
bash fonts/build.sh
```

**Адаптив.** Точки: 1240px (прячется блок «Работаем»), 1100px (бургер-меню),
900px (в шапке остаётся только телефон), 760px и 480px (сетка карточек).
Горизонтального скролла нет ни на одной ширине от 360px до 1440px+.

**Изображения в WebP**, все в 2× от размера отображения. Фото и рендеры карт —
`-q 90 -m 6 -sharp_yuv` (PSNR 43–50 dB, визуально неотличимо от исходных PNG),
иконки форматов — lossless. Суммарно 6.8 MB → 0.70 MB.
Исключение — `logo.png`: используется в `<link rel="icon">`, где PNG совместимее.

```bash
cwebp -q 90 -m 6 -sharp_yuv -metadata none in.png -o out.webp
```

## Известные особенности

- В шрифте Jost нет глифа `№` — в футере он подставляется из системного шрифта.
  Так же ведёт себя и оригинальный макет.
- Высота страницы 8754px против 8708px во фрейме Figma. Расхождение не в вёрстке:
  в макете высоты нескольких секций не сходятся с их же auto-layout (например,
  фрейм «Нестандартные карты» — 486px при содержимом 510px). Свёрстано по
  auto-layout.
